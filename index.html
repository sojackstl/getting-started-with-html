What is the internet?
The internet is a series of communications between a person, a pc, a connection and servers. When a person types in a website link, like CNN it sends a HTTP request to a server. The server then returns to the user with the appropriate HTML file so that the user can view this document.

HTML
HTML is the language of the internet. It is made up of several components. The first is the text content, which is what you see. The second is the markup, which is what it looks like. The third is references to other pages. The fourth and final piece is references to other websites.

Markup
Markups are tags that make webpages look the way they do. Tags looks like < name > content < / name > and when entered correctly effect the way that the user views the page. One type of tag is the tag for bold, it looks like < b > . Another such tag is the one for italics, that tag looks like < em > < / em >. It is important to note that tags end with a "/".

HTML Attributes
These allow you to add things to your HTML document that the user sees. The first type of attribute is those that allow links to be used. This tag looks like < a href=>. It allows you to link websites like ESPN to your document. Another attribute is that for images. This, as the name would suggest, allows you to put images onto your webpage. They can also create an imaginarty "box" around text with the < p > tag that has a height and width. These are called "block" elements.
